# Project 5 — ISO 27001 Audit Readiness Programme

## Overview
Led ISO 27001 audit preparation at CyBlack and New Horizon, maintaining the Statement of Applicability, conducting internal pre-audit assessments, and assembling evidence packs.

## Statement of Applicability (SoA)

Maintained a live SoA for all 93 ISO 27001:2022 Annex A controls documenting:
- Applicability (Yes / No)
- Justification for inclusion or exclusion
- Implementation status (Implemented / Partial / Not Implemented)
- Evidence reference or implementing policy document

## Internal Audit Programme

Conducted 3 months before external audit:
- Reviewed control design adequacy against Annex A requirements
- Tested operational effectiveness through sampling and observation
- Documented nonconformities and opportunities for improvement
- Assigned corrective actions with owners and target dates
- Verified closure before external audit date

## Evidence Categories

| Category | Evidence Examples |
|---|---|
| Access Management | Access review records, MFA configs, provisioning logs |
| Vulnerability Management | Nessus reports, patch records, remediation evidence |
| Incident Response | Incident tickets, post-incident reviews, playbooks |

| Security Awareness | Training records, phishing simulation results, policy acknowledgements |
| Risk Management | Risk register, treatment plans, risk acceptance sign-offs |
