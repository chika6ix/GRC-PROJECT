# KPI Definitions — Compliance Monitoring Framework

## Access Management KPIs

| KPI | Target | Source |
|---|---|---|
| MFA Adoption Rate | >95% | Identity Provider logs |
| Privileged Account Review Completion | 100% | Access review records |
| Orphaned Account Count | 0 | HR + Active Directory |
| Policy Exception Count | <5 | Exception register |

## Vulnerability Management KPIs

| KPI | Target | Source |
|---|---|---|
| Critical CVE Remediation Rate (48hrs) | 100% | Nessus / Qualys |
| High CVE Remediation Rate (7 days) | >95% | Nessus / Qualys |
| Patch Compliance % | >98% | Patch management tool |
| Vulnerability Recurrence Rate | <5% | Vulnerability scanner |

## Incident Response KPIs

| KPI | Target | Source |
|---|---|---|
| Mean Time to Detect (MTTD) | <4 hours | SIEM |
| Mean Time to Respond (MTTR) | <2 hours | Incident tickets |
| SLA Adherence Rate | >99% | Incident management system |
| Post-Incident Review Completion | 100% | PIR register |
