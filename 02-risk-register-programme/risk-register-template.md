# Risk Register Template

| Field | Description |
|---|---|
| Risk ID | Unique reference e.g. R001 |
| Risk Title | Short descriptive name |
| Risk Description | Full description of the risk event |
| Asset / Process | What is at risk |
| Threat Source | External / Internal / Third Party / Natural |
| Likelihood (1-5) | 1=Rare, 5=Almost Certain |
| Impact (1-5) | 1=Negligible, 5=Critical |
| Inherent Risk Score | Likelihood x Impact |
| Existing Controls | Controls currently in place |
| Control Effectiveness | Effective / Partially Effective / Ineffective |
| Residual Risk Score | Risk remaining after existing controls |
| Risk Appetite Threshold | Organisation's defined threshold |
| Treatment Option | Mitigate / Accept / Transfer / Avoid |
| Treatment Actions | Specific steps to treat the risk |
| Risk Owner | Accountable person |
| Target Date | Completion date for treatment |
| Residual Risk After Treatment | Expected score post-treatment |
| Review Date | Next scheduled review |
| Status | Open / In Progress / Closed / Accepted |
